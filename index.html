Sentinel

A request filtering and bot-detection system built on top of Spring Security — uses a custom scoring engine combined with AI-driven decision logic for ambiguous traffic.

  • Built a custom OncePerRequestFilter to intercept and evaluate every incoming request
  • Implemented static scoring logic: 0–20 → Allow, 60–100 → Block
  • Integrated AI-based decision engine for mid-score traffic (21–59) to reduce false positives
